# Nevada Tracker — Archived

This is the original **Great Pennington Migration** house-tour tracker, built for the
Las Vegas / Henderson search (April 2026). The family is now moving to Estrella Mountain
Ranch in Goodyear, Arizona, so this project is retired.

It is kept here intact, not deleted. Everything still works as a self-contained unit:
open `index.html` (desktop) or `mobile.html` (phone). It carries its own
`manifest.json`, `sw.js`, icons, `data/properties.json`, and `photos/`.

Nothing in here is wired into the new Arizona site.
